# Демонстрация стратегий редукции лямбда-термов
## Реализованно
* Веб интерфейс
* Парсер программ на языке из входного файла (на данный момент из ввода пользовaтеля в веб приложении)
* Пошаговое выполнение редукций и их демонстрация
* Обработка ошибок
* Юнит-тесты
